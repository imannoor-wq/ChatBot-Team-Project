# Normes de Col·laboració de l'Equip
Per garantir un flux de treball correcte, el nostre equip segueix aquests estàndards:

1. **Flux de GitHub:** Utilitzem GitHub Classroom per documentar tots els canvis i garantir la traçabilitat del projecte.
2. **Missatges de Commit:** Cada actualització del codi ha de tenir una descripció clara del canvi realitzat.
3. **Distribució de Tasques:** El treball es divideix de manera equitativa entre tots els membres del grup.
4. **Qualitat:** Mantenim la documentació actualitzada en format Markdown impecable.
